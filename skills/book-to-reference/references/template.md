Use this template when creating reference files from book content.

---

```markdown

From "{Full Book Title}"

## Core Concept

> **{One-sentence definition of the core concept}**

{2-3 sentences expanding on the definition}

## Key Ideas

### {Subtopic 1}

{Explanation with bullet points}

- Point 1
- Point 2
- Point 3

### {Subtopic 2}

{Use tables for comparisons}

| Aspect | Description | Example |
|--------|-------------|---------|
| ... | ... | ... |

## Frameworks/Models

### {Model Name}

## Key Mantras

- "{Quote 1}" — {Context}
- "{Quote 2}" — {Context}
- "{Quote 3}" — {Context}
```

---

## Guidelines

### Length
- Target: 200-400 lines per file
- If topic is larger, split into multiple files

### Style
- Use imperative/infinitive form for instructions
- Prefer tables over long paragraphs
- Use code blocks for models/frameworks
- Include page numbers or chapter references when useful

### What to Include
- Core definitions
- Actionable frameworks
- Memorable quotes
- Concrete examples

### What to Exclude
- Lengthy anecdotes (summarize instead)
- Redundant explanations
- Content that requires images to understand
- Opinions without actionable insight

### Naming Convention
```
{author-lastname}-{topic-keyword}.md

Examples:
- sylvester-elegance.md
- rouse-player-psychology.md
- schell-lenses.md
- koster-fun.md
```
