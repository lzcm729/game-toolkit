# 技能深度与可达性 (Depth)

**理论基础**: Sylvester 技能理论（深度/可达性/再发明/弹性挑战）、Schell Lens #34 技能、Lens #38 挑战、Lens #54 可达性

**关注点**:
- 技能天花板是否足够高（高手仍有提升空间）
- 技能门槛是否足够低（新手不会被挡在门外）
- 游戏是否在玩家变强时"再发明自己"（操作期→策略期→心理期）
- 是否有弹性挑战（同一机制允许不同程度的成功）
- 失败处理是否惩罚玩家本人（重做/等待）还是惩罚角色（损失/挫折）
- 是否存在失败陷阱（玩家被锁在注定失败的状态中）

**Prompt 模板**:
```
从【技能深度与可达性】视角评审此设计文档（基于 Sylvester 技能理论）。重点关注：
- 技能天花板：精通系统后是否仍有有意义的提升空间？
- 技能门槛：新手首次接触时的学习曲线是否合理？
- 再发明：随着玩家变强，系统是否呈现不同层次的挑战？（操作→策略→心理）
- 弹性挑战：同一机制是否允许不同程度的成功（而非全对/全错）？
- 失败处理：失败时玩家付出的代价是什么？是否惩罚了玩家本人而非角色？
- 失败陷阱：是否存在玩家被锁在必定失败状态而无法脱出的场景？
- 隐式难度调节：不同玩家策略是否自然对应不同难度？
提出具体的深度/可达性问题和修复建议，标注严重程度（🔴高/🟡中/🟢低）。
```
