# 决策质量与信息平衡 (Decision)

**理论基础**: Sylvester 决策理论（信息平衡、决策范围、心流杯模型）、Schell Lens #39 有意义的选择

**关注点**:
- 玩家在关键决策点是否有足够（但不过多）的信息
- 决策结果是否"部分可预测"（既非随机也非必然）
- 决策范围变化是否合理（有无连续的非决策或不可能决策）
- 是否存在信息饥饿或信息过载
- 心流间隙：是否有玩家大脑"无事可嚼"的空白期
- 不同决策是否有范围变化（闪电反应/战术推敲/深度思考交替）

**Prompt 模板**:
```
从【决策质量与信息平衡】视角评审此设计文档（基于 Sylvester 决策理论）。重点关注：
- 每个决策点的信息平衡：玩家知道多少？需要推测多少？答案是否显而易见？
- 决策结果的可预测性：是否"部分可预测"？结果能否从机制逻辑推导？
- 决策范围变化：是否混合了不同规模的决策（快速/战术/深层）？
- 信息饥饿场景：玩家是否会面对"完全不知道该怎么选"的困境？
- 信息过载场景：是否存在"答案显而易见，不需要思考"的假决策？
- 心流间隙：操作之间是否有大脑无事可做的空白期？
引用 Sylvester 的"有意义的决策基于非显而易见地使用以显而易见的方式运作的机制"原则。
提出具体的决策质量问题和修复建议，标注严重程度（🔴高/🟡中/🟢低）。
```
