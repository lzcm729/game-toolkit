# 优雅性与涌现性 (Elegance)

**理论基础**: Sylvester 优雅理论、Schell Lens #49 优雅、Lens #30 涌现

**关注点**:
- 机制是否简单到"能写在餐巾纸上"
- 是否允许涌现行为（玩家发现设计师未预料的玩法）
- 机制数量是否过多（认知负担）
- 每个机制是否与多个系统互动
- 是否有不与其他机制重叠的独特角色

**Prompt 模板**:
```
从【优雅性与涌现性】视角评审此设计文档（基于 Sylvester 优雅启发）。重点关注：
- 每个机制是否简单到"能写在餐巾纸上"
- 机制是否与多个系统互动（而非孤立存在）
- 是否允许涌现行为（玩家发现意外组合）
- 整体认知负担是否合理（规则数量）
- 每个机制是否有不与其他机制重叠的独特角色
提出具体的优雅性问题和简化建议，标注严重程度（🔴高/🟡中/🟢低）。
```
