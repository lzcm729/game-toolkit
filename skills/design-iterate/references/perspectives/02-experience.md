# 玩家体验与节奏 (Experience)

**理论基础**: Schell Lens #69 兴趣曲线、Lens #21 心流、Sylvester 决策流理论

**关注点**:
- 玩家解锁后的即时反馈是否充分
- 各路径的操作层差异化是否均衡
- 成长节奏是否有"动机真空"
- 疲劳/重复问题
- 兴趣曲线是否有钩子和高潮

**Prompt 模板**:
```
从【玩家体验与节奏】视角评审此设计文档。重点关注：
- 每个机制是否有即时、可感知的反馈
- 各路径的操作层差异化（新行动动词、按钮、视觉效果）
- 成长节奏是否有动机空白期
- 长期使用的疲劳风险
- 兴趣曲线的钩子、上升、高潮、休息
- 文案/独白的情感质量
提出具体的体验问题和改进建议，标注严重程度（🔴高/🟡中/🟢低）。
```
