# 动机与强化循环 (Motivation)

**理论基础**: Sylvester 动机理论（强化计划/置换效应/奖励对齐）、Schell Lens #46 奖励、Lens #47 惩罚、Lens #23 动机

**关注点**:
- 奖励计划是否有效（固定比率 vs 变量比率 vs 叠加计划）
- 外在奖励是否置换了内在乐趣
- 奖励对齐：奖励是否鼓励玩家做他们本来就想做的事
- 是否有叠加的强化计划（多个奖励流同时运行）
- "再来一回合"综合症：是否有足够多的即将到来的奖励让玩家不舍得停下
- 玩家悔恨风险：机制是否有驱动但不满足的倾向

**Prompt 模板**:
```
从【动机与强化循环】视角评审此设计文档（基于 Sylvester 动机理论）。重点关注：
- 奖励计划类型：是固定比率（可预测、易疲劳）还是变量比率（持续吸引）？
- 置换效应：给本身有趣的核心玩法加外在奖励是否会削弱内在乐趣？
- 奖励对齐：奖励系统鼓励的行为和玩家自然想做的行为是否一致？
- 叠加计划：在任何时刻，是否有至少一个奖励即将触发（避免动机真空）？
- 长期动机弧线：从第1天到第25天，驱动玩家继续的动力是什么？是否有衰退？
- 玩家悔恨风险：系统是否让玩家投入大量时间后感觉"被骗"而非"被满足"？
- 内在 vs 外在：哪些乐趣来自系统本身的趣味，哪些依赖外挂奖励？
提出具体的动机/强化问题和修复建议，标注严重程度（🔴高/🟡中/🟢低）。
```
