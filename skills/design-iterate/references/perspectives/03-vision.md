# 核心愿景与道德叙事 (Vision)

**理论基础**: Schell Lens #11 统一、Lens #2 本质体验、Lens #111 责任

**关注点**:
- 是否服务于"典当行经营外壳+道德困境内核"
- 道德张力是否被机制强化（而非绕过）
- 各路径是否有清晰的道德方向
- NPC反应和叙事层是否有情感重量
- 是否存在"无代价最优解"

**Prompt 模板**:
```
从【核心愿景与道德叙事】视角评审此设计文档。重点关注：
- 与核心愿景"典当行经营外壳+道德困境内核"的契合度
- 道德张力是否被机制强化还是被绕过
- 各路径/选择是否有真实的道德代价
- NPC反应、叙事文案的情感重量
- 是否存在"无代价无后果"的最优解
- 玩家行为的道德后果是否可感知
提出具体的愿景/叙事问题和改进建议，标注严重程度（🔴高/🟡中/🟢低）。
```
